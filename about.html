About this project

Lambda Build Week

This was one of the project that was build in Lambda School Build Week. Students and Project Lead collaborated together using Github, Slack and Trello to complete this project.

React App

There is a sign in and sign up function build in this project. User can create log in to the account and link to a personal dashboard.

Other Features

In the user dashboard user can fill out the forms for ailment which they need help for and the build in strains recommender will provided the recommended strain.
